# Security Policy

ZeroKey AI is a local-first static web app. It has no backend, no server-side storage and no telemetry.

## Important notes

- Prompts are sent directly from the browser to the selected AI provider.
- Optional API keys are stored in the browser's `localStorage`.
- Do not enter sensitive secrets, passwords, private keys, medical data or confidential business data.
- Use GitHub Pages or another HTTPS host when publishing.

## Reporting issues

Open a GitHub issue with:

- Browser and device.
- Steps to reproduce.
- Expected result.
- Actual result.
- Console error, if any.

Do not include real API keys in reports.
